## Features

- typescript
- jest
- prettier
- husky
- eslint

## Usage

```sh
npm i -g yo @pickel-project/generator-node-lib
```

```sh
yo @pickel-project/node-lib
```

## Development

```sh
git clone https://github.com/pickel-project/generator-node-lib.git
yarn
yarn run build
yarn link
```
