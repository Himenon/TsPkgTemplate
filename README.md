# generator-node-lib

Node library template generator.

## Features

- typescript
- jest
- prettier
- husky
- eslint

## Usage

```
npm i -g yo @pickel-project/generator-node-lib
```

```
yo @pickel-project/generator-node-lib
```

## Development

```sh
git clone https://github.com/pickel-project/generator-node-lib.git
yarn
yarn run build
yarn link
```
